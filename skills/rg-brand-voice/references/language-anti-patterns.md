# Language Anti-Patterns

Use this for voice audits, final polish, and AI-tell cleanup.

This reference is adapted for RaidGuild from repo-level `references/writing-anti-patterns.md`.

## Hard Rewrite Terms

Rewrite the thought when these appear:

- leverage
- synergy / synergize
- revolutionary
- game-changing
- cutting-edge
- best-in-class
- seamless / seamlessly
- empower
- unlock, when used as vague value language
- supercharge
- next-generation / next-gen
- paradigm shift

## AI Tells

Challenge these:

- delve / dive into
- "In today's fast-paced world"
- "In the ever-evolving landscape"
- "It's not just X, it's Y"
- "It's important to note"
- "It's worth noting"
- "Let's unpack"
- "Picture this"
- "Imagine a world where"
- "What if I told you"
- "Whether you're a seasoned expert or just getting started"
- "At the end of the day"
- "Hope this helps"

## RaidGuild-Specific Failures

- fake guild voice that hides the actual update
- occult texture without receipts
- public copy that exposes workflow machinery
- agent persona speaking as a human or official HQ account
- broad claims about community, clients, or governance without source support

## Structural Tells

- repeated three-item list rhythm
- colon headlines that sound like generic thought leadership
- symmetric paragraph lengths
- summative closers
- bullet lists where prose would be clearer
- unsupported hedges such as "many believe" or "it is widely known"

## Audit Pass

1. Search for hard rewrite terms.
2. Search for phrase tells.
3. Check whether the draft starts with a generic opener.
4. Check whether lore obscures the factual point.
5. Vary rhythm where the draft sounds machine-regular.
6. Rewrite the thought, not only the flagged word.
