# Workflow Patterns

Use this to convert Prism workflows into reusable content recipes without turning each workflow into a skill.

## Extract From Each Workflow

- input source
- content job
- channel
- approval gate
- output artifacts
- publishing destination
- safety checkpoint
- reusable strategy pattern

## Common Patterns

### Draft Review Publish

Use for blog posts, newsletters, Portal posts, and public updates.

Sequence:

1. intake
2. source ledger
3. strategy brief
4. draft
5. voice pass
6. safety pass
7. human review
8. publish ops
9. reconcile live URL

### Field Notes

Use for firesides, sessions, community activity, and research packets.

Sequence:

1. gather source pack
2. identify 2-4 angles
3. choose one primary public angle
4. produce follow-on briefs for blog, wiki, social, or video

### Queen Raida Social

Use for agent-account updates.

Sequence:

1. source facts
2. Queen Raida angle
3. short social format
4. voice pass
5. safety pass
6. human approval unless workflow explicitly allows automatic publication

### Public Brief Or Podcast

Use for periodic synthesis.

Sequence:

1. period intake
2. top signals
3. public/private split
4. brief script
5. media plan if needed
6. safety review
7. publish/reconcile
