# Create Report

Use `rg-content-strategy` and `rg-public-output-safety` for public-facing or Discord-bound text.

Create `portal-activity-report.md` with:

- time window
- headline metrics
- notable sessions/content/activity
- arcade/game stats when requested
- feedback themes when requested
- notable gaps or source failures
- concise Discord-ready summary

Avoid private contact details unless the destination is explicitly internal and the source permits it.
