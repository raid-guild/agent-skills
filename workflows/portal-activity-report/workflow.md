# Portal Activity Report

This workflow creates Portal activity reports from Portal CMS data, feedback records, and game/arcade reporting APIs.

It replaces standalone reporting skills with a workflow recipe. Use `rg-portal-ops` for Portal facts and source APIs, `rg-content-strategy` for report structure, `rg-public-output-safety` for public-safe summaries, and `rg-publishing-ops` for Discord or queued output.
