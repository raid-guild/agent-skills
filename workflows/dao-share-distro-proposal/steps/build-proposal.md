# Build Proposal

Use `rg-dao-ops`.

Prepare the DAO proposal payload or unsigned transaction artifact from `share-distro-table.json`.

Produce `dao-proposal-payload.json` and `dao-proposal-review-notes.md` with exact DAO, target chain, proposal title/body, actions, values, calldata or command, and verification checklist.

Do not submit onchain in this step.
