# Brief Plan

Use `rg-content-strategy`.

Read `brief-source-ledger.md` and produce `brief-plan.md` with:

- audience
- visibility
- output set
- story spine
- section outline
- priority highlights
- facts to avoid or keep internal
- media requirement: `none`, `podcast`, `video`, or `podcast_and_video`
- Portal target collection or resource type, if known

For `fireside_recap`, plan around the completed session and select a small set of evidence-backed highlights rather than a broad community roundup.
