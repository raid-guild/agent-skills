# Publish Prep

Use `rg-portal-ops`, `rg-public-output-safety`, and `rg-publishing-ops`.

Prepare the exact publish payload or queue payload.

Include:

- destination collection or channel
- visibility
- final brief text
- media URLs, if any
- title and description
- approved source links
- public sanitizer result for public outputs
- Discord-ready or X-draft summary when requested

Produce `publish-payload.md` or `publish-payload.json`, matching the destination schema when known.
