# Script Or Skip Media

Use `rg-brand-voice` and `rg-media-render-ops`.

Read `brief-plan.md` and `brief-draft.md`.

If media is not required, mark the route as `briefOnly` and produce `media-skip.md` with the reason.

If media is required, produce `media-script.md` with:

- host roles and voices
- spoken script
- title and description candidates
- lower-third or scene text if needed
- approved source references for claims
- intended composition or render type, if known

Default host names may follow the existing Prism convention when appropriate:

- Queen Raida: main host
- The Jester: secondary host

Route:

- `media` -> `review`
- `briefOnly` -> `review`
