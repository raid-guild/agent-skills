# Publish

Use `rg-portal-ops` and `rg-publishing-ops`.

This agent node has `autoRun: false`. Continue only after the operator approves the exact destination, visibility, and payload.

Publish only the approved payload. Do not change visibility, title, description, or public copy except for destination-required formatting.

Capture:

- created or updated Portal record
- live URL or draft URL
- media URL
- Discord/X queue result, if any
- sanitizer result for public outputs
- request/workflow reference

Produce a final publish summary with evidence links and any follow-up needed.
