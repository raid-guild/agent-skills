# Strategy Brief

Use `rg-content-strategy`.

Create `strategy-brief.md`.

Include:

- source mode
- audience
- channel/destination
- content job
- source facts
- missing facts
- primary angle
- voice layer
- recommended format and length
- media recommendation
- review gates

Mode-specific guidance:

- `generic`: clarify the strongest source-backed public point.
- `steered`: follow the approved steering comment as source of truth.
- `fireside`: keep the post narrow and thesis-driven; do not write a broad recap.
- `lore`: plan scene, continuity, symbols, conflict, and public-safe grounding before drafting.

If source facts are too thin, create `strategy-blocker.md` and stop with missing facts.
