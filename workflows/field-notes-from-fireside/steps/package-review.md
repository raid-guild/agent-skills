# Package Review

Human gate. Review the source pack, topic map, media brief, and Portal artifact update.

Approval means the package is good enough to:

- create one Bard Calendar X draft for later review/scheduling
- create queued child requests for blog, wiki, and recap media work

Approval does not mean publish to X, publish Portal content, render video, or auto-run any child workflow.

Route:

- `approved` -> `calendar-draft`
- `changesRequested` -> `topic-map`
- `rejected` -> `closed`
