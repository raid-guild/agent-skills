# Source Pack

Use `rg-content-strategy` and `rg-portal-ops`.

Build a verified source pack from session artifacts and Portal data.

Produce `fireside-source-pack.md` with:

- direct session evidence
- outside context, if used
- public-safe facts
- internal-only or uncertain notes
- direct quotes only when short and appropriate
- source URLs or artifact ids
- gaps and candidate follow-up questions

Do not fill holes with plausible inference. When a gap appears, name the gap and list distinct possibilities.
