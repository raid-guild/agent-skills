# Media Brief

Use `rg-content-strategy` and `rg-brand-voice`.

Create `fireside-media-brief.md` for downstream image/video/recap work.

Include:

- visual direction
- recap video angle
- useful source moments
- candidate title and description direction
- reusable image/video concepts
- constraints and banned claims

Default visual direction should stay in RaidGuild fireside mode: bonfire, strategy table, maps, source cards, runes, guild craft, ember light, and fellowship. Translate modern topics through fantasy objects and composition. Avoid generic office, SaaS dashboard, and stock-business imagery unless explicitly requested.

Do not create a Portal post, draft article prose, write wiki pages, render video, or publish anything.
