# Repo Work

Use `rg-content-strategy`, `rg-brand-voice`, `rg-public-output-safety`, and Prism built-in `target-deploy-ops`.

This agent node has `autoRun: false` because it modifies a target repository.

Follow the target repo's `AGENTS.md` or equivalent local instructions before editing. Create the article markdown artifact, implement the repo change, validate locally, and open or update a PR when repo tooling permits.

Do not include secrets, private Discord content, client-private details, internal operational notes, or unsupported claims.
