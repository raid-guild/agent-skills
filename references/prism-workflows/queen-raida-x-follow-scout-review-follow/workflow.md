# Queen Raida X Follow Scout Review Follow

This workflow discovers candidate X accounts for Queen Raida to follow, ranks them, waits for human review, and follows only approved accounts. It is account-shaping behavior, so discovery and following are separated by a human gate.

Use Prism Knowledge, the Queen Raida docs source, Portal CMS profile data, and X API context. Treat follows as public social graph changes. Never auto-follow from discovery alone.
