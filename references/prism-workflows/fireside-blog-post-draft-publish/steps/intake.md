# Production Intake

Identify source mode: `session-topic` or `mature-wiki-topic`.

Required inputs:
- selected topic
- desired thesis or question
- Portal session id and/or wikiPage id/slug
- target length
- publication destination
- required links or constraints

If the topic is broad, route back to wiki development instead of drafting a blog. Verify source objects directly through Portal/Prism APIs when possible. Do not infer missing session, wiki, or Portal facts.

Create or update source notes for later steps and state whether the request is ready for `draft-post`.
