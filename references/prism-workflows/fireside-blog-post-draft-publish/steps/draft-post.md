# Draft Post

Draft one focused blog post from the selected topic.

Use rg-scribe-publisher-skill for RaidGuild voice. Keep the post narrow, thesis-driven, and source-backed. Default target length is 600-900 words unless the request says otherwise.

Requirements:
- one clear thesis
- no broad fireside recap
- no unrelated session themes used as filler
- no invented facts or unsupported Portal claims
- source links preserved where appropriate
- clear header image direction for the next step
- note any inline charts, maps, or diagrams that would help the article; these may be represented as inline SVGs later

Create `blog-draft.md` with the draft and source notes. Return the artifact name and one-sentence thesis.
