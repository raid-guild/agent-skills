# Intake

Turn the request description, comments, and attachments into a durable lore brief.

Use Prism retrieval when useful. Prioritize Queen Raida and RaidGuild context. Recommended searches include:
- Queen Raida persona
- Queen Raida media inspiration
- RaidGuild lore
- the requested theme, place, project, proposal, or motif

Create a request artifact named `lore-intake.md` with:
- requested premise or seed
- desired audience and public channel, if known
- real context that may inspire the fiction
- forbidden facts, privacy concerns, or unsupported claims to avoid
- tone target: serialized lore / short story, not newsletter
- required CMS tag: `lore`
- open questions or blockers, if any

If the prompt is too vague, make conservative assumptions and record them. Do not ask follow-up questions unless there is a hard safety or publishing blocker.

Return the artifact name, the core premise, and whether drafting can proceed.
