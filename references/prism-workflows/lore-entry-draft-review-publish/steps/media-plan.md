# Media Plan

Plan image generation for the lore entry.

Read `draft.md` and `lore-arc.md`. Create a request artifact named `media-plan.md` with:
- featured image concept
- optional inline image concepts
- visual motifs and continuity elements
- image prompts suitable for generation
- negative prompt / avoidance notes
- recommended alt text
- where each image should appear in the post

Image style should support the fiction: cinematic, emblematic, artifact-like, or worldbuilding-rich. Avoid generic fantasy stock imagery, cluttered logos, unreadable text in images, or anything that implies real unapproved claims.

Return the artifact name and the selected image concepts.
