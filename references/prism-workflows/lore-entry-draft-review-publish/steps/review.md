# Review

Human review gate.

Approve only if:
- the post reads as fictional lore, not a newsletter
- the lore is grounded enough in Queen Raida / RaidGuild context to feel native
- public safety checks pass
- image artifacts fit the story and are suitable for publishing
- CMS tagging will include `lore`

Routes:
- approved -> publish-prep
- changesRequested -> revise
- rejected -> closed
