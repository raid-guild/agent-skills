# Generate Images

Generate the images planned in `media-plan.md` using the available image generation capability.

Save generated image files or image artifacts on the request whenever the runtime path supports it. If direct image artifact storage is unavailable, create an image manifest artifact that includes generated file paths, prompts, and intended placements.

Minimum output:
- one featured image candidate
- alt text
- prompt provenance in `media-plan.md` or a generated image manifest

Return the generated image artifact names or manifest name and any limitations.
