# Publish Prep

Prepare the final CMS package.

Read approved `draft.md`, `media-plan.md`, generated image artifacts, and review notes.

Create `publish-package.json` or `publish-package.md` with:
- title
- slug recommendation
- dek / excerpt
- body content
- selected featured image and inline image placements
- alt text
- tags, including `lore`
- category recommendation, if applicable
- SEO title and description
- reviewer notes / public-safety checks

The package must explicitly include the `lore` tag. Return the artifact name and readiness summary.
