# Internal Summary

## Outcome

Create an internal written brief for members using the resolved date window.

## Use This Context

- `brief-window.json`
- relevant digests, artifacts, and notes from the chosen window
- any internal audience guidance from the request

## Durable Artifacts

Write both:
- `internal-summary.md`
- `internal-summary.json`

The JSON artifact should include:
- date window
- main themes
- notable internal updates
- unresolved items or watchpoints when appropriate

## Rules

- optimize for member usefulness, not public polish
- do not include raw secrets or credentials
- if the window was expanded, make that explicit in the summary
