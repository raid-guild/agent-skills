# Media Plan

Read the latest draft artifact for this request and propose image needs for the post. Create two request artifacts:

- image-plan.json
- image-prompts.md

The plan should identify whether a hero image is needed and which inline images, if any, add editorial value. Limit inline image proposals to at most 2.

image-plan.json should include a compact machine-readable structure for the requested hero and inline images, including purpose, placement, style notes, and whether each item is required or optional.

image-prompts.md should contain human-readable prompts suitable for image generation.

## Inline image rule

Only propose inline images when they add real editorial value inside the article body. For each inline image, include a clear placement hint tied to a paragraph or section so the publish step can embed it in the final CMS content instead of uploading it without placement.
